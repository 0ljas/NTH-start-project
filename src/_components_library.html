Библиотека компонентов проекта

Демонстрация внешнего вида компонентов, из которых собран проект. Расположение блоков на этой странице регулируется глобальной модульной сеткой и, если она отключена (что хорошо для большинства проектов), блоки смещаются друг под друга.

Проектные блоки — те, что cозданы специально для этого проекта (по умолчанию их нет вовсе).

Исходные блоки — те, что есть в стартовом наборе блоков проекта (все они отключены по умолчанию).

Исходный блок: Модульная сетка

Классы .container, .row, .col-[xs|sm|md|lg|xl|xxl]-[1 - @grid-columns]. См. глобальную сетку и примеси.
